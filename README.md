# FlexWake (智能调休闹钟)

FlexWake 是一款专为中国工作制量身定制的 iOS 智能调休闹钟应用。它能够无缝处理 5.5 天工作日（周一至周六）的作息，并根据中国政府每年公布的法定节假日与调休补班安排，自动智能调整响铃。

---

## 🌟 核心功能

1. **5.5天智能调休**：专为周一至周六工作制打造，自动跳过法定放假日，并在调休补班日（例如周补班的周日）自动响铃唤醒。
2. **7日排程时间轴**：首页顶部展示未来一周的排程状态，每日是否响铃及“休/班”标注一目了然。
3. **下次响铃倒计时**：首卡显示精准响铃倒计时（例如“距离下次响铃还有 8 小时 15 分钟”），提供即时心安反馈。
4. **自定义规则库**：内置官方补班与放假表，并支持随时手动添加临时自定义放假/补班日期。
5. **月度网格日历**：调休测试页提供极简手绘日历，在日期下方直观标注“休”与“班”，通览全月状态。
6. **完善的闹钟设置**：支持稍后提醒 (Snooze)、铃声选择及硬件触觉震动等丰富功能。
7. **数据安全与隐私**：纯本地运行，严格保障您的数据安全与个人隐私。

## 🚀 快速上手与真机运行

### 环境要求
- **iOS 部署目标**：iOS 17.0+
- **编译工具**：Xcode 15.0+

### 部署步骤
1. 在 Mac 上使用 Xcode 打开 `TiaoxiuAlarm.xcodeproj`。
2. 用数据线将您的 iPhone 手机连接到 Mac。
3. 在 Xcode 顶部 Scheme 栏中，将目标运行设备从 Simulator 改为您的 **iOS 物理手机**。
4. 点击主工程，进入 **Signing & Capabilities** 面板，确保勾选了您的个人证书（Personal Team）。
5. 按下快捷键 **`Cmd + R`** 启动编译，即可在手机中安装并体验 **FlexWake**。
